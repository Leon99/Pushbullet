Pushbullet
==========
- Portable .NET library for Pushbullet (https://www.pushbullet.com)
  - Supports all the operations allowed by the API
  - Supported platfroms:
    - .NET Framework 4.5
    - Windows Store apps
    - Windows Phone 8
  - Some basic unit tests
- Console app that uses the library. Enables pushes from a CLI.
